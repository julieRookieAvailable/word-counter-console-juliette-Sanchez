# Word Counter Console Application

## Author
Juliette Sánchez

## Description
This is a Java console application that counts the occurrences of each word in a paragraph entered by the user.  
The project demonstrates:

- **Object-Oriented Programming (OOP)** principles
- **SOLID design principles**
- **Separation of concerns** (Input, Processing, Output)
- **Centralized logging** instead of `System.out.println` in logic classes
- **Unit testing** with JUnit
- **Proper exception handling** and validation

---

## Features

- Accepts a paragraph of text as input from the console
- Counts all unique words (case-insensitive)
- Displays each word with the number of occurrences
- Handles empty input gracefully
- Centralized logging for info and error messages
- Unit tests cover happy path and edge cases
